1. Dashboard
    1.1 Основана страница
        Лог със действията на потребителите към целия проект.  Те ще се групират.
        Ще има Avatar на потребител. Неговото име. Дата. И списък на действията.
        Няма да има paging а ще има infinite scroll
        
    1.2 Sidebar
        1.2.1 Списък с хората, които участват в проекта - 2 варианта
            - в1 списък с аватари, при минаване с мишката над всеки се появява тултип с име, последно активен
            - в2 аватари с име и последно активен вдясно от тях
            - като се мине над потребител да се highlight-ват неговите действия
        1.2.2 Статистика
            - PieChart задачи ( изпълнени / отхвърлени / оставащи )
    
2. Section
    2.1 Основана страница
        2.1.1 Име на секцията
            иконка /img/icons.png
            име на секцията
            - когато се мине с мишката на полето се показват 2 бутона - промени и изтрий
            Промяна на заглавието с ефект и появяването на форма за неговата промяна
            
        2.1.2 Съдържание на секцията
            Тя ще съдържа списък с неща (items) - tasks / dividers / notes и други
            когато се минава между двa item-a ще се появява бутон "add here >" и след като се натисне той ще се появява панел подобен на 2.1.3

            Типове съдържание:
            
            2.1.2.1 Task 
                - на 2 реда
                - 3 side checkbox (empty / finished / rejected)
                - Името на отговорния за задачата човек
                - Текста на самата задача (ако е повече от N символа, ще се показва само часто от текста със връзка ... (more) )
                - Дата на задачата 
                - (ако) задачата е за текущия потребител е с по-разлчен цвят
                - (ако) задачата е изпълнена става по-бледа
                - (ако) задачата е отказана става по-бледа и задраскана
                - (ако) е изпълнена или отхвърлена до checkbox-а има иконка archive ( т.е. само ри класове closed/rejected ще се вижда класа .archive )
                - (ако) е архивирана, иконката става unarchive
                - като се мине с мишката над задачата се показват бутоните
                 - горе в дясно - иконка за влачене ( с възможност за преместване на задача в друга секция или за друг човек)
                 - долу в дясно - промени / коментари ( брой ) / подзадачи ( брой ) / прикачени файлове ( брой ) / изтрии, като ако броя на нещо е 0, няма да се показва
                - (действие) изтриване - изтрива се със ефект задачата 
                 - динамично се променя статистиката по страницата
                - (действие) промени
                 - на мястото на задачата се поява форма за промяна на задача
                - форма за добавяне / промяна на Task 
                    - текста на задачата на няколко реда
                    - кой е отговорен за задачата ( да ли да ми се прати писмо )
                    - прикачени файлове
            
            2.1.2.2 Divider
                служи като разделител на съдържанието
                Той ще може да се затвяря, като при затваряне ще скрива всички неща между него и следващия divider
                когато е затворен
                    - ще има брой на item-ите който са в него
                като се мине с мишката върху него се появят иконки за местне / триене / промяна 
                форма за добавяне / промяна 
                    - input[type=text] за името разделителя
                
            2.1.2.3 Notes [ TODO ]
            
            2.1.2.4 Files [ TODO ]
            
        
        2.1.3 Footer на секцията, ще съдържа връзки
            - добави task
            - добави divider
            - добави note
            - добави файл(ове)
        
        
    2.2 Sidebar
        2.2.1 Списък с хората, който участва в тая секция (като при 1.2.1)
            - toggable
            - по default ще е затворено
        2.2.2 Покажи задачите само за <select>.
            - в <select> - me / anyone / anyone except me + хората очастващи в проекта
            - toggable
            - по default ще е затворено
        2.2.3 Статистика
            - брои задачи / изпълнени задачи / отказани задачи / архивирани
            - да има покажи / скрий архивираните
        2.2.5 Export & Import
            - toggable
            - по default ще е затворено
            - changelog за секцията (export)
            - rss формат (export)
            - print версия
    
3. Task
    3.1 Основана страница
        3.1.1 Task
            - 3 side checkbox (empty / finished / rejected)
            - пълния текст на задачата
            - дата на задачата
            - за кого е задачата
        3.1.2 Comments
            Avatar, дата, текст, ако има добавени под задачи от този коментар, ако има добавени файлове от коментара
            възможност за промяна/изтриване в следващите 15 минути.
        3.1.3 Add Comment
            Поле за коментара ( ще се разпъва само при повече текст )
            2 бутона -  more (който става less) | Add Comment
            При натискана не на more се появя с ефект разширентата форма ( и бутона more става less)
                - Files поле
                - Subtasks поле
        
    3.2 Sidebar
        2.2.1 Subtasks
            - 3 side checkbox (empty / finished / rejected)
            - като се мине с мишката имаме - edit / delete / drag
        2.2.2 Attachments
            - път до файла
            - download
            - delete
        2.2.3 Списък с хората, който участва в тая задача (като при 1.2.1)
